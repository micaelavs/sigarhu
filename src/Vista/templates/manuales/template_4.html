    • INTRODUCCIÓN

    La Dirección de Informatica y Tecnologia del Ministerio de Transporte de la Nación ha diseñado, desarrollado e implementado un sistema a pedido de la Dirección General de Recursos Humanos que permite Administrar todos los datos que definen al Empleado, Segmentando la Administración y actualización de los mismos, a nivel de datos según responsabilidades de cada una de las direcciones que administran los mismos (Administración, Desarrollo y Control de Gestión, Planificación y Convenios).

    ACCESO AL SISTEMA

    Para acceder al Sistema de Gestión y Administración de RRHH, el usuario deberá ingresar a la siguiente dirección: {{{SITIO}}} y despues ingresar al modulo SIGARHU.

    Para ingresar al sistema, el usuario deberá usar su usuario y contraseña de dominio, una ves ingresado los datos podra navegar segun las credenciales que posea.

    LEGAJOS

    POR AGENTE

    Al acceder al sistema automaticamente nos redirecciona al listado de agentes en donde figuran todos los agentes que hay actualmente hasta la fecha además podemos filtrar las busquedas por los campos DEPENDENCIAS, ESTADO, MODALIDAD DE VINCULACIÓN o SITUACIÓN DE REVISTA.
    Tambien se pueden visualizar por la cantidad de registros de 10, 25, 50 o hasta 100 además el sistema incorpora un buscador.

    Podemos observar que tambien figura el botón EDITAR.
    Para modificar los datos del agente hacemos click en el botón EDITAR, que se encuentra en la columna "Acciones", y podemos modificar los datos de dicho agente.
    Al seleccionar EDITAR se pueden modificar las siguientes solapas, ya que el resto de solapas solo se puede visualizar la información.

    • SITUACIÓN ESCALAFONARIA
    • PERFILES DE PUESTOS
    • FORMACIÓN
    • ANTIGUEDAD

    Como dato adicional tambien se puede agregar observaciones y agregar, editar y visualizar la documentación del legajo.

    Al ingresar al exportador de datos, el sistema nos ofrece exportar los campos que necesitamos de cada solapa del legajo del agente además de aplicar filtros de ESTADO, MODALIDAD DE VINCULACIÓN, SITUACIÓN DE REVISTA, POR DEPENDENCIA, POR NIVELES y POR DEPENDENCIAS SUBORDINADAS.
    Al seleccionar los campos que necesitamos, el sistema descargara un archivo Excel en el que visualizaremos los datos de los campos solicitados.

    DATOS PERSONALES

    La sección Datos Personales es la primera de todo el ABM de agentes. Aqui se visualizaran los datos del agente y el usuario de este rol solo puede visualizar la información y no editarla.

    SITUACIÓN ESCALAFONARIA

    En esta sección se van a poder modificar los campos AGRUPAMIENTO, FUNCIÓN O NIVEL, TRAMO, GRADO y COMPENSACIÓN TRANSITORIA además de habilitar la opción DELEGACIÓN GREMIAL para eso hacemos click en Modificar Escalafón Actual y el sistema nos permitira modificar dicho campo.

    UBICACIÓN DE ESTRUCTURA

    Actualmente esta sección es de solo lectura, por lo que no es posible modificar un dato.

    PERFILES DE PUESTOS

    En esta sección se pueden modificar los siguientes campos.

    • PUESTO
      • Familia de Puestos
      • Nombre del Puesto
      • Nivel de Destreza
      • Puesto Supervisa
    • ACTIVIDADES/TAREAS
      • Nivel de Complejidad de las Tareas
    • EVALUACIONES
      • Cargar Evaluaciones
    EVALUACIONES

    Dentro del apartado de Evaluaciones podremos cargar las evaluaciones del empleado seleccionando el boton Nuevo.

    Deberemos completar todos los campos obligatorios y tambien podremos adjuntar un archivos .PDF
    Luego de Guardar la informacion podremos visualizarla en la pantalla de Historial y poder ver el Comprobante adjunto.

    FORMACIÓN

    En esta sección se completa la formación academica del agente, se debe completar todos los campos de los grupos TITULOS OBTENIDOS, OTROS ESTUDIOS REALIZADOS y CONOCIMIENTO ESPECIFICO DE SISTEMAS/SOFTWARE.
    Luego procedemos a Guardar para aplicar los cambios.
    En el apartado de Titulos Obtenidos a cada titulo cargado se le podran asignar creditos, esto se realiza seleccionado el boton de Porcentaje al lado de cada titulos.
    Tambien podremos marcar uno de los titulos como Titulo Considerado.

    Seleccionando el boton de Porcentaje ingresaremos en la siguiente pantalla.

    Seleccionando el boton Nuevo accederemos a la pantalla para cargar los creditos.
    En esta misma pantalla tambien podran modificarse una vez cargados.

    Tambien dentro del apartado de Formacion tendremos la seccion de Cursos Sistema Nacional de Capacitacion en la que podremos agregar los cursos del empleado.
    Cuando se carguen mas de 10 cursos se podran visualizar dentro de un Historial.

    ANTIGÜEDAD

    Solamente se puede modificar el campo FECHA OTORGAMIENTO GRADO en esta sección.

    POR CUIT

    Al ingresar a esta sección el sistema nos ofrece un campo para ingresar el CUIT del empleado, esto sirve para buscar un empleado de manera facil, rapida y accesible.

    Al ingresar el CUIT del agente el sistema nos redirecciona a la gestión de legajos de dicho agente donde se puede editar los datos según el rol.

    CREDITOS INICIALES

    En este apartado podremos cargar los Creditos Iniciales para el empleado.

    Seleccionando el boton Nuevo podremos acceder al alta de los creditos iniciales.

    ADMINISTRACIÓN DE RECURSOS

    En esta sección se centra en el alta, baja y modificación de datos mediante distintos tipos de ABM de todas las solapas.

    TITULOS

    Este ABM permite agregar, borrar y editar opciones en el campo TITULOS.
    Tambien disponemos de un buscador y mostrar entre 10, 25, 50 o 100 registros.

    ALTA DE TITULOS

    Para realizar un alta solamente se debe hacer click en NUEVO y el sistema nos redireccionara para realizar el alta.
    Para realizar el alta debemos ingresar el TIPO TITULO, ABREVIATURA y NOMBRE DEL TITULO. Luego hacer click en GUARDAR.

    CREDITOS PARA PROMOCION

    Este ABM permite dar de alta los creditos con su correspondiente Tramo y Nivel.

    Para dar de alta los creditos deberemos seleccionar el boton Nuevo y completar todos los campos obligatorios.

    FAMILIA DE PUESTOS

    Este ABM permite agregar, borrar y editar opciones en el campo FAMILIA DE PUESTOS.
    Tambien disponemos de un buscador y mostrar entre 10, 25, 50 o 100 registros.

    ALTA DE FAMILIA DE PUESTOS

    Para realizar un alta primero se debe hacer click el botón NUEVO y el sistema nos redireccionara para realizar el alta.
    En el alta debemos ingresar el NOMBRE FAMILIA DE PUESTO. Despues hacer click en GUARDAR para finalizar.

    SUBFAMILIA DE PUESTOS

    Este ABM permite agregar, borrar y editar opciones en el campo SUBFAMILIAS DE PUESTOS.
    Tambien disponemos de un buscador y mostrar entre 10, 25, 50 o 100 registros.

    ALTA DE SUBFAMILIAS DE PUESTOS

    Para realizar un alta primero se debe hacer click el botón NUEVO y el sistema nos redireccionara para realizar el alta.
    En el alta debemos ingresar el FAMILIA DE PUESTO y SUBFAMILIA. Despues hacer click en GUARDAR para finalizar.

    PUESTOS

    Este ABM permite agregar, borrar y editar opciones en el campo PUESTOS.
    Tambien disponemos de un buscador y mostrar entre 10, 25, 50 o 100 registros.

    ALTA DE PUESTOS

    Para realizar un alta solamente se debe hacer click en NUEVO y el sistema nos redireccionara para realizar el alta.
    Para realizar el alta debemos ingresar el SUBFAMILIA y PUESTO y despues hacer click en GUARDAR para finalizar.

    INFORMES

    AUDITORIA

    En este apartado podremos consultar los datos que sean necesarios de cualquier agente en especifico.
    Todos los cambios realizados dentro del sistema a nivel de datos se veran reflejados en el apartado de Auditoria.

    Deberemos seleccionar el icono de la columna Acciones para poder visualizar los datos y la accion realizada para cada empleado.

    IMPORTADOR

    IMPORTAR CURSOS

    Dentro de este apartado podremos cargar los cursos que se visualizaran dentro del sistema mediante formato Excel.

    CERRAR SESIÓN

    Para salir del sistema el usuario debe CERRAR SESIÓN en el botón que se encuentra en la esquina superior derecha.
